# PDD Armenia

Веб-приложение для подготовки к экзамену по правилам дорожного движения Армении.

## Возможности

- 1032 вопроса по 10 темам (маневрирование, знаки, перекрёстки, безопасность и др.)
- Экзамен: 20 вопросов, 30 минут, допускается не более 2 ошибок
- Статистика по каждому вопросу и теме
- Избранные вопросы и работа над ошибками
- Тёмная/светлая тема, настройка размера шрифта
- Полностью клиентское приложение, данные хранятся в localStorage

## Стек

- React 18, TypeScript
- Material UI 5
- React Router 6
- SASS, CSS Modules

## Запуск

```bash
yarn install
yarn start
```

## Сборка

```bash
yarn build
```
